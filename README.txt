Discord Username Sniper by KHALID

This project contains:
- backend.py (Python sniper tool)
- index.html, style.css, script.js (simple frontend)

How to run:

1. Install dependencies:
   pip install requests colorama flask

2. Run backend:
   python backend.py

3. Open index.html in your browser to access UI.

Note: The frontend sends requests to backend which needs to be run separately.

Enjoy sniper tool by KHALID!
